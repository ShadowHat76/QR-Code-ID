As this application queries a database for validation, it needs a MySQL database
This is a mini guide to remind myself how to convert from xlsx to a database

0. Download and install MySQL

1. Convert the xlsx file into a CSV, this can be done with any Spreadsheet viewer.
2. Open the CSV file with a text editor and remove random commas that may have been autogenerated.
3. Use MySQL to create a database and table
    sudo mysql
    >CREATE <database_Name>;
    >USE <database_Name>;
    >CREATE TABLE <table_Name>(<field_Name> <Data_Type>(<Data_Length>));

# Example: CREATE myDB
# Example: CREATE TABLE Users(ID int, Name varchar(100), Email varchar(100));

4. Copy the CSV file into the Secure File Directory; Identify this directory by running
    >SHOW VARIABLES LIKE "secure_file_priv";

5. Load the Data into the Database using
    >LOAD DATA INFILE '<file_Name>.csv' INTO TABLE <database_Name>.<table_Name> FIELDS TERMINATED BY ',';

6.  Remember to ALTER the Database Column and ADD CONSTRAINT(s) as needed using mysql commands before deployment
